# Next.js with OpenAI Integration

This project demonstrates the integration of OpenAI's models with a Next.js application, leveraging the capabilities of Node.js 18. It is designed to showcase how AI-powered features can be embedded within a React-based framework.

## Features

- **Next.js**: A React framework for production.
- **OpenAI API**: Integration of various OpenAI models to enhance application capabilities.

## Prerequisites

Before running this project, make sure you have the following installed:
- Node.js 18.x
- npm (usually comes with Node.js)
